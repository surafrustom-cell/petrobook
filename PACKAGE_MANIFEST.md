## HOW TO CREATE THE ZIP

### Windows (File Explorer)
1. Open the `READY_FOR_GITHUB` folder
2. Press **Ctrl+A** to select all files
3. Right-click → **Compress to ZIP file** (Windows 11) or **Send to → Compressed folder**
4. Name it: `petrobook-v1.1.0.zip`

> **Important:** Make sure hidden files are visible before selecting all.
> In File Explorer: View → Show → Hidden items

### Mac (Finder)
1. Open the `READY_FOR_GITHUB` folder
2. Press **Cmd+A** to select all files
3. Press **Cmd+Shift+.** first to show hidden files (`.firebaserc`, `.gitignore`, `.github/`)
4. Right-click → **Compress X items**
5. Rename the result to `petrobook-v1.1.0.zip`

### Terminal (any OS)
```bash
cd /path/to/parent-folder

# Mac/Linux:
zip -r petrobook-v1.1.0.zip READY_FOR_GITHUB \
  --exclude "*/serviceAccount.json" \
  --exclude "*/node_modules/*" \
  --exclude "*.bak" \
  --exclude "*/.firebase/*"

# Verify contents:
unzip -l petrobook-v1.1.0.zip | head -40
```

### Windows PowerShell
```powershell
# Navigate to parent of READY_FOR_GITHUB
cd C:\Users\YourName\Desktop

# Create ZIP (excludes serviceAccount and node_modules)
Compress-Archive -Path .\READY_FOR_GITHUB\* -DestinationPath .\petrobook-v1.1.0.zip

# Verify
Expand-Archive -Path .\petrobook-v1.1.0.zip -DestinationPath .\verify-temp -Force
Get-ChildItem .\verify-temp -Recurse | Select-Object FullName
Remove-Item .\verify-temp -Recurse
```

When you (or someone else) unzips this on a new computer:

2. Edit `index_standalone.html` — paste real `FIREBASE_CONFIG` values
3. Edit `.firebaserc` — paste real project ID
4. Follow `DEPLOY_NOW.md` — exact commands to go live
